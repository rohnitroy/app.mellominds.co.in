# MelloMinds — Database Schema Reference

> Read-only reference. Do NOT edit this file to change the DB — use migration scripts.
> Use `full_migration.sql` to recreate this schema on any new PostgreSQL / PGAdmin instance.

---

## Tables Overview

| Table | Purpose |
|---|---|
| Plans | Subscription plan definitions |
| Users | Therapist accounts (email + Google OAuth) |
| Calendars | Booking service types created by therapists |
| Appointments | All booked sessions |
| Availability | Therapist weekly schedule |
| Clients | Client profiles per therapist |
| ClientTransfers | Transfer requests between therapists |
| ClientActivities | Activities/homework assigned to clients |
| SessionNotes | Session notes written by therapists |
| NoteTemplates | Custom note templates per therapist |
| UserIntegrations | OAuth tokens + payment gateway credentials |
| Notifications | In-app notifications |
| enterprise_leads | Enterprise enquiry form submissions |

## Key Relationships

```
Users (therapist)
  ├── Calendars (1:many)
  │     └── Appointments (1:many)
  ├── Availability (1:many — weekly schedule)
  ├── Clients (1:many)
  │     ├── ClientActivities (1:many)
  │     └── ClientTransfers (many:many via from/to therapist)
  ├── SessionNotes (1:many via therapist_id)
  ├── NoteTemplates (1:1)
  ├── UserIntegrations (1:many — google + cashfree)
  └── Notifications (1:many)
```
